# Roadmap Archive

This directory stores historical roadmap snapshots.

Rules:

- `ROADMAP.json` is the only active roadmap source of truth.
- Markdown roadmap files in this directory are historical snapshots or supporting notes.
- Archive filenames use ISO dates for lexical ordering.

Examples:

- `2026-04-03-roadmap-foundation.md`
- `2026-04-03-roadmap-strategic.md`
